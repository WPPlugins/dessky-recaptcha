Protect your website against spam bots with reCAPTCHA.

== Description ==

Dessky reCAPTCHA is the lightweight WordPress plugin for website protection against spam comments and brute-force attacks. As name suggests it is protecting your website by adding Google's reCAPTCHA to the login, comment, registration and forgot password forms.

This plugin was developed by [Dessky](http://dessky.com/).

= What is protected with Google reCAPTCHA? =

* Comment form
* Forgot password form
* Registration form
* Login form

You can select in admin where reCAPTCHA will be required.


= Credits =

Dessky reCAPTCHA is based on the 'Simple Google reCAPTCHA' plugin developed by [Minor](https://profiles.wordpress.org/minor/).


== Installation ==

1. Upload the plugin files to the "/wp-content/plugins/dessky-recaptcha" directory, or install the plugin through the WordPress plugins screen directly.

2. Activate the plugin through the "Plugins" screen in WordPress.

3. Use the Settings => Dessky reCAPTCHA admin page to configure the plugin.
